# 玉铃的音乐宇宙

一个充满爱意的音乐探索网页应用，包含星空动画、音乐播放器和私人信件功能。

## 功能特点

- ✨ **星空探索**：点击星星解锁不同的音乐
- 🎵 **音乐播放器**：在线播放精选歌曲
- 📝 **个人笔记**：记录喜欢的歌词和心情
- 💌 **私人信件**：一封特别的信
- 🔒 **密码保护**：专属的私人空间

## 如何使用

1. 直接在浏览器中打开 `玉铃的音乐宇宙.html` 文件
2. 输入密码进入应用
3. 点击星星开始探索音乐宇宙
4. 可以在音乐播放时添加个人笔记

## 技术实现

- HTML5 + CSS3 + JavaScript
- Canvas 星空动画
- 响应式设计
- 本地存储（笔记功能）

## 开发说明

### 启动本地服务器

```bash
# 使用Python启动服务器
python -m http.server 8000

# 或者使用PowerShell脚本
.\start_server.ps1
```
